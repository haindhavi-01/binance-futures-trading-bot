# Binance Futures Trading Bot

## Features

- MARKET Orders
- LIMIT Orders
- Logging
- Error Handling
- Binance Futures Testnet Integration

---

## Setup

Install dependencies:

pip install -r requirements.txt

---

## Run

python cli.py

---

## Example Input

Enter Symbol: BTCUSDT
Enter Side: BUY
Enter Order Type: MARKET
Enter Quantity: 0.001
